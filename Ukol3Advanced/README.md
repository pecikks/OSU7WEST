# Ukol3Advanced

This project is a simple car showroom gallery built with HTML and CSS.

The page displays a grid of car cards with images, model names, short descriptions, and "new" badges on selected cars. It uses `galerie.html` for the page structure, `styles.css` for styling, and the `auto*.jpg` files as gallery images.

To view the project, open `galerie.html` in a web browser.
