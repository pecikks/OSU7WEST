# Ukol5NeuAdvanced

This is a simple static website for a fictional 2026 car exhibition focused on Volkswagen AG models.

## Project Contents

- `index.html` - homepage with the car gallery and main navigation
- `program.html` - exhibition schedule with times, activities, and locations
- `registration.html` - visitor registration form with basic input validation
- `style.css` - styles, responsive layout, gallery cards, forms, tables, and light/dark theme styling
- `auto1.jpg` to `auto8.jpg` - car images used in the gallery
- `pattern-1px.png` - small background pattern used by the page styling

## How to Run

Just open `index.html` in a browser to view the project.
