# Ukol4Advanced

A simple web gallery for a Volkswagen AG car exhibition.

The project displays an introductory hero section with the exhibition program and a responsive car gallery. Each model is shown in a card with a photo, name, short description, hover effect, and a badge for new models.

## Project Contents

- `galerie.html` - main HTML page of the gallery
- `styles.css` - styles, responsive layout, and animations
- `auto1.jpg` to `auto8.jpg` - car photos
- `pattern-1px.png` - background pattern used in the hero section

## How to Run

The project does not require any server. Just open `galerie.html` in a browser.
